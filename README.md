# BitcoinPrice

Fetches Bitcoin price from Coindesk api.

##Initalisation

First import the libary, then initiate like so:

```
Bitcoin = BitcoinPrice("Currency shorthand")
```

The currency shorthand is left blank it will default to USD

Currencies:

- USD
- EUR
- GBP

- UpdatePrice()
